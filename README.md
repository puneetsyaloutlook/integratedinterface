# Integrated Workspace — ITI prototype

Interactive wireframe prototype for the Integrated Task Interface (ITI). Built as a single-file HTML prototype for feasibility workshop use.

## Screens

- **Home** — actions snapshot and claims snapshot
- **Actions** — overdue action queue with detail panel
- **Claims** — task metrics, portfolio dashboard, and claims list
- **Operations** — team analytics and agent analytics
- **Claim detail** — Motor #4821 scenario with snapshot, documents, timeline, and stages tabs

## Deploying

Deployed automatically via Vercel on push to `main`.
